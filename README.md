The psc package implements a method to perform personalised synthetic controls. 
This method is used to compare the outcomes observed in a Data Cohort (DC) 
against the counter-factual evidence obtained from a statistical model - the 
Counter Factual Model (CFM).  

It is envisaged here that all information on the experimental arm is available 
through the data (DC) and all information on the control arm is available 
through the model (CFM).  The psc method then allows for the measurement of any
difference between a dataset and a model.

The main function of the psc package is 'pscfit()' which needs two inputs, 
a model object and a dataset.  The model objects supported are 'glm' for binary,
count and continous outcomes and 'flexsurvreg' for survival outcomes.  
Of importance is that the DC supplied contains covariates which match the 
names of the covariated included in the CFM.

Once fitted, standard 'summary()' and 'plot()' commands can be used to evaluate 
the fit.  'plot_ite()' can also be used to investigate the individual effects for 
each patient.


## Installation

You can install the development version of psc from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("richJJackson/psc")
```

## Example

For the basic example we use example models and datasets included in the psc
package.  

### Load model

The model generated on simulated patients based on the 
data used to generate the PROSASH model is used and can be loaded from the psc 
package (REF: "[Using prognostic and predictive clinical features to make 
personalised survival prediction in advanced hepatocellular carcinoma patients 
undergoing sorafenib treatment. Berhane S, et al., Br J Cancer. 2019 
Jul;121(2):117-124](https://doi.org/10.1038/s41416-019-0488-4)")

``` r
library(psc)
surv.mod <- psc::surv.mod
surv.mod
```

### Fitting model

The 'pscfit()' function is used to compare the two sources of information

```r
surv.psc <- pscfit(surv.mod,data)
```

### Evaluating fit

Standard 'summary()' function will evaluate the fit...

``` r
summary(surv.psc)
```

and plots can using plot to see a plot of what you have done

```
plot(surv.psc)
```
